GITHUB:

VERSION CONTROL:
   - Version control is apractice of tracking and managing changes to software code.
   - This helps teams to work faster,
        . Reduces development time and increses successful deployment.
        . Keeps track of every modification in the code.
        . If mistake is made, developers can track back to the previous version and compare it.
        . Helps prevent mistake.
